# 智能车辆导航系统

一个专为移动端设计的大气车辆导航网页应用。

## 功能特性

- 📍 **实时定位**: 获取当前位置坐标
- ⚡ **时速显示**: 实时显示行驶速度
- 🚦 **交通信息**: 展示当前路况和预计到达时间
- 🎨 **大气界面**: 渐变背景和毛玻璃效果
- 📱 **移动端适配**: 完美适配各种移动设备

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 渐变)
- JavaScript (ES6+)
- Geolocation API

## 使用方法

1. 在浏览器中打开 `index.html`
2. 授权位置访问权限
3. 点击"开始导航"按钮

## 文件结构

```
.
├── index.html      # 主页面
├── style.css       # 样式文件
├── script.js       # 功能脚本
└── README.md       # 项目说明
```

## 浏览器兼容性

- Chrome 60+
- Firefox 55+
- Safari 11+

## 注意事项

- 需要HTTPS环境才能正常使用地理定位功能
- 在移动设备上使用体验最佳
- 确保设备开启了定位权限
